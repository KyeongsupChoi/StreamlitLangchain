# Voice Agent

Build a voice-enabled agent that listens to speech, processes it with a LangChain agent, and speaks the response back. This guide covers the full pipeline: Speech-to-Text (STT), agent processing, and Text-to-Speech (TTS).

---

## Architecture Options

### STT > Agent > TTS ("Sandwich" Architecture)

The most common pattern processes audio in three discrete stages:

1. **STT**: Convert user speech to text (e.g., AssemblyAI, Whisper, Deepgram)
2. **Agent**: Process the text with a LangChain agent (standard text-based agent)
3. **TTS**: Convert the agent's text response to speech (e.g., Cartesia, ElevenLabs, OpenAI)

This approach reuses existing text-based agents and is the most flexible.

### Speech-to-Speech

End-to-end models that directly process audio input and produce audio output (e.g., GPT-4o audio, Gemini multimodal). Fewer components but less control over individual stages.

This guide focuses on the sandwich architecture for maximum flexibility and modularity.

---

## Producer-Consumer Pattern for STT

Real-time voice applications use a producer-consumer pattern: the STT service produces text as the user speaks, and the agent consumes complete utterances.

### AssemblyAI Real-Time STT

```python
import assemblyai as aai
import queue

aai.settings.api_key = "your-assemblyai-api-key"

# Thread-safe queue for passing transcripts between STT and agent
transcript_queue = queue.Queue()

def on_transcript(transcript: aai.RealtimeTranscript):
    """Callback fired when a transcript segment is ready."""
    if not isinstance(transcript, aai.RealtimeFinalTranscript):
        return  # Ignore partial transcripts
    text = transcript.text.strip()
    if text:
        transcript_queue.put(text)

def on_error(error: aai.RealtimeError):
    """Handle STT errors."""
    print(f"STT Error: {error}")

# Create the real-time transcriber
transcriber = aai.RealtimeTranscriber(
    sample_rate=16_000,
    on_data=on_transcript,
    on_error=on_error,
)
```

### Microphone Input

```python
import pyaudio

SAMPLE_RATE = 16_000
FRAMES_PER_BUFFER = 3200

audio = pyaudio.PyAudio()
stream = audio.open(
    format=pyaudio.paInt16,
    channels=1,
    rate=SAMPLE_RATE,
    input=True,
    frames_per_buffer=FRAMES_PER_BUFFER,
)

transcriber.connect()

try:
    while True:
        audio_data = stream.read(FRAMES_PER_BUFFER, exception_on_overflow=False)
        transcriber.stream(audio_data)
except KeyboardInterrupt:
    pass
finally:
    transcriber.close()
    stream.stop_stream()
    stream.close()
    audio.terminate()
```

## LangChain Agent with Streaming

The agent processes transcribed text and streams its response for low-latency TTS.

```python
from langchain_openai import ChatOpenAI
from langgraph.prebuilt import create_react_agent
from langchain.tools import tool
from langgraph.checkpoint.memory import MemorySaver

@tool
def get_weather(city: str) -> str:
    """Get the current weather for a city."""
    # Replace with actual weather API call
    return f"The weather in {city} is sunny and 72 degrees."

@tool
def get_time(timezone: str = "UTC") -> str:
    """Get the current time in a timezone."""
    from datetime import datetime
    return f"The current time in {timezone} is {datetime.now().strftime('%I:%M %p')}."

llm = ChatOpenAI(model="gpt-4o", temperature=0)
checkpointer = MemorySaver()

agent = create_react_agent(
    model=llm,
    tools=[get_weather, get_time],
    prompt="You are a helpful voice assistant. Keep responses concise and conversational.",
    checkpointer=checkpointer,
)
```

### Streaming Agent Responses

```python
def get_agent_response(user_text: str, thread_id: str) -> str:
    """Get a complete response from the agent."""
    config = {"configurable": {"thread_id": thread_id}}
    result = agent.invoke(
        {"messages": [{"role": "user", "content": user_text}]},
        config=config,
    )
    return result["messages"][-1].content


async def stream_agent_response(user_text: str, thread_id: str):
    """Stream agent response token by token."""
    config = {"configurable": {"thread_id": thread_id}}
    async for event in agent.astream_events(
        {"messages": [{"role": "user", "content": user_text}]},
        config=config,
        version="v2",
    ):
        if event["event"] == "on_chat_model_stream":
            token = event["data"]["chunk"].content
            if token:
                yield token
```

### Conversation Memory

The checkpointer automatically maintains conversation history per `thread_id`. Each voice session uses a unique thread ID so the agent remembers the conversation context.

```python
import uuid

session_id = str(uuid.uuid4())

# First turn
response1 = get_agent_response("What's the weather in Paris?", session_id)
# "The weather in Paris is sunny and 72 degrees."

# Second turn - agent remembers context
response2 = get_agent_response("What about London?", session_id)
# "The weather in London is sunny and 72 degrees."
```

## Text-to-Speech with Cartesia

Cartesia provides low-latency streaming TTS suitable for real-time voice applications.

```python
from cartesia import Cartesia

cartesia_client = Cartesia(api_key="your-cartesia-api-key")

VOICE_ID = "a0e99841-438c-4a64-b679-ae501e7d6091"  # Example voice ID
MODEL_ID = "sonic-2024-10-19"

def text_to_speech(text: str) -> bytes:
    """Convert text to speech audio bytes."""
    audio_data = cartesia_client.tts.bytes(
        model_id=MODEL_ID,
        transcript=text,
        voice_id=VOICE_ID,
        output_format={
            "container": "wav",
            "encoding": "pcm_s16le",
            "sample_rate": 24000,
        },
    )
    return audio_data
```

### Streaming TTS

For lower latency, stream TTS output as the agent generates text:

```python
def stream_text_to_speech(text_stream):
    """Convert a stream of text chunks into streaming audio."""
    audio_stream = cartesia_client.tts.sse(
        model_id=MODEL_ID,
        transcript=text_stream,
        voice_id=VOICE_ID,
        stream=True,
        output_format={
            "container": "raw",
            "encoding": "pcm_s16le",
            "sample_rate": 24000,
        },
    )
    for chunk in audio_stream:
        if chunk.get("audio"):
            yield chunk["audio"]
```

## Complete Pipeline Composition

Connect all three stages using `RunnableGenerator` for a streaming pipeline:

```python
from langchain_core.runnables import RunnableGenerator

async def stt_producer(audio_stream):
    """Produce text from an audio stream."""
    async for audio_chunk in audio_stream:
        transcriber.stream(audio_chunk)
        # Check for completed transcripts
        while not transcript_queue.empty():
            yield transcript_queue.get()


async def agent_processor(text_stream):
    """Process text through the agent and yield response chunks."""
    async for user_text in text_stream:
        async for token in stream_agent_response(user_text, session_id):
            yield token


async def tts_producer(text_stream):
    """Convert text stream to audio stream."""
    buffer = ""
    async for token in text_stream:
        buffer += token
        # Send to TTS when we have a complete sentence
        if buffer.endswith((".", "!", "?", "\n")):
            audio = text_to_speech(buffer.strip())
            yield audio
            buffer = ""
    # Flush remaining buffer
    if buffer.strip():
        audio = text_to_speech(buffer.strip())
        yield audio


# Compose the full pipeline
stt = RunnableGenerator(stt_producer)
agent_step = RunnableGenerator(agent_processor)
tts = RunnableGenerator(tts_producer)

pipeline = stt | agent_step | tts
```

## WebSocket Endpoint

Deploy the voice agent as a WebSocket service for real-time bidirectional communication:

```python
from fastapi import FastAPI, WebSocket
import uvicorn

app = FastAPI()

@app.websocket("/voice")
async def voice_endpoint(websocket: WebSocket):
    """WebSocket endpoint for real-time voice interaction."""
    await websocket.accept()
    session_id = str(uuid.uuid4())

    try:
        while True:
            # Receive audio bytes from client
            audio_bytes = await websocket.receive_bytes()

            # STT: Convert audio to text
            transcriber.stream(audio_bytes)

            # Check for completed transcripts
            while not transcript_queue.empty():
                user_text = transcript_queue.get()
                print(f"User said: {user_text}")

                # Agent: Process the text
                response_text = get_agent_response(user_text, session_id)
                print(f"Agent response: {response_text}")

                # TTS: Convert response to audio
                audio_response = text_to_speech(response_text)

                # Send audio back to client
                await websocket.send_bytes(audio_response)

    except Exception as e:
        print(f"WebSocket error: {e}")
    finally:
        await websocket.close()


if __name__ == "__main__":
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

### Client-Side Connection

```javascript
const ws = new WebSocket("ws://localhost:8000/voice");

// Send audio from microphone
navigator.mediaDevices.getUserMedia({ audio: true }).then(stream => {
    const recorder = new MediaRecorder(stream);
    recorder.ondataavailable = event => {
        ws.send(event.data);
    };
    recorder.start(100); // Send chunks every 100ms
});

// Receive and play audio response
ws.onmessage = event => {
    const audioBlob = new Blob([event.data], { type: "audio/wav" });
    const audioUrl = URL.createObjectURL(audioBlob);
    const audio = new Audio(audioUrl);
    audio.play();
};
```
